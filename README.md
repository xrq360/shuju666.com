已经关闭，无demo，感兴趣可自行搭建。部分截图可看 [数据牛推特](https://twitter.com/shuju666)

# 系统结构
网站设计：使用注册制查询，用户免费注册后可查询大部分数据。用户提交其他没有的数据，审核后可申请vip，查询所有数据。

系统开发框架：thinkphp 3.2.3

数据存储：ElasticSearch

用户存储：MySQL

硬件配置：Vultr运营商，16G内存，1T硬盘

设备支持：中文版、英文版、移动端
